Boid Draw

This is an attempt to produce natural-looking, sketchy lines from the clumsy movement of the user's mouse. The path of the mouse creates a course that a few of boids attempt to follow. The algorithms are based on Craig Reynolds' path following algorithms via Daniel Shiffman. The 2D Vector class used is derived from Dan Shiffman's 3D version for the Processing library. (Note: Only tested on Chrome and Firefox, Win XP. Let me know if something isn't working.)
